# ADR-0005: 跨境界 stat 修真 ladder（×1.5 / ×2 / ×3 / ×5 / ×10）

每次大境界突破时，HP / 灵力 / 战斗伤害 等核心 stat 全部按**修真 ladder** 倍率跳跃：

| 突破 | 倍率 |
|---|---|
| 炼气 → 筑基 | × 1.5 |
| 筑基 → 金丹 | × 2 |
| 金丹 → 元婴 | × 3 |
| 元婴 → 化神 | × 5 |
| 化神 → 大乘 | × 10 |

最晚期（大乘 9/9）vs 早期（炼气 1/9）= **450 倍** HP 差。

## 为什么

修真经典里"境界差 = 数量级差"——元婴老怪碾压金丹晚辈是常识。固定 ×2 太均匀，反而失去"修真 ladder"质感。非线性 ladder 让每一次大境界突破都明显"变强了"，后期 boss 战才显得有压力。

## 考虑过的方案

- **固定 × 2** — 简单，均匀，但缺修真感
- **修真 ladder（本方案）** — 非线性，每境界倍率不同，最修真
- **按 stat 分** — HP × 2 / 伤害 × 3 / 灵力 × 1.5 — 调优空间大但复杂
- **自定曲线** — 让 dev 写数值表

## 后果

- 后期数值巨大（HP 4.5 万 → 45 万），combat / Boss 数值要按 ladder 调
- 越后期越要靠**金手指 修为 ×N** 跳过闭关（30 分钟一次）
- 程序化生成（装备/丹药/灵兽）的品级曲线也按 ladder 走
- 平衡表只写一份（炼气 1/9 基准），其他按倍率推
