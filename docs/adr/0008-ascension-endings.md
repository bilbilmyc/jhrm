# ADR-0008: 飞升 ending 由道心决定（4-6 个结局）

玩家达到 **大乘 9/9** → 触发飞升 IF 段 → 飞升结局由**道心**决定：

| 道心 | 飞升身份 |
|---|---|
| 剑道 | 剑仙 |
| 魔道 | 魔尊 |
| 王道 | 圣王 |
| 隐道 | 散仙 |
| 无道 | 天地客 |
| ... | ... |

共 **4-6 个 endings**。

## 为什么

道心是玩家整个游戏的 meta 选择（每次 IF 选都在投"道心票"），**最终回报应该在 ending**。多 ending 给 replay 价值——玩家换个道心再玩一局，看不同结局。

## 考虑过的方案

- **单一飞升 ending** — 简单，缺 replay
- **道心决定（本方案）** — 4-6 ending，与"道心是 meta 选择"对齐
- **道心 + 因果** — 8-10 ending，更细但工作量大
- **新游戏+：传承重玩** — 完全不同的 mechanic（保留到 v0.2+ 候选）

## 后果

- **道心是全游戏最重要的隐式选择**：每次 IF 决策都会推/拉道心值
- **4-6 个飞升 IF 段**需要写（每段 ~200-500 字）
- 因果不影响 ending（按本方案）—— 善恶只影响 IF 段内 NPC 态度，不决定最终身份
- 如果以后想做"道心 + 因果"双维度 ending，只需扩 ADR，不重写

## 跟 MVP 的关系

MVP（炼气期）**不含飞升 IF**。但 MVP 必须能从一开始**追踪道心**——所有 IF 选项都附带 `heart:` 标签，统计后决定大乘期的飞升身份。这意味着 MVP 的 IF 段设计要预留道心统计字段。
