# 声望 (Reputation) 设计

> **Date**: 2026-06-25
> **Status**: DEFERRED to 筑基+ (per `docs/decisions.md` 决定 #15：字段保留 forward-compat，7 派系/任务/福利 延后实现)
> **Unlock**: 筑基期

---

## 1. 概述

声望衡量玩家在 7 个江湖派系中的身份地位。每个派系独立声望轴，范围 -100（敌对）到 +100（传说）。派系之间存在互斥关系：与正道亲近会疏远魔道，反之亦然。

---

## 2. 数据模型

```dart
enum Faction { 蜀山派, 魔宗, 妖族, 散修, 朝廷, 灵草谷, 坊市 }

class Reputation {
  Map<Faction, int> factionRep;            // -100 ~ +100 per faction
  Map<Faction, FactionRank> ranks;          // 当前等级
  List<String> titles;                      // 已解锁称号
}

enum FactionRank {
  素人,        // 0
  初识,        // 10
  小有名气,    // 25
  名声鹊起,    // 50
  一方之霸,    // 75
  传说,        // 100
}
```

## 5. 声望获取

- 派系任务完成：+10 ~ +30
- 派系见证下杀敌：+5
- 跨境界突破：所有派系 +5
- 渡劫成功：所属派系 +10
- 因果大善：善派 +10，恶派 -10
- 因果大恶：恶派 +10，善派 -10

---

## 7. 派系福利

- **派系商店**：声望 25 解锁，可购买派系专属法宝、丹药
- **派系任务**：声望 50 解锁，奖励丰厚
- **派系同门**：声望 75 可召唤同门协助战斗
- **派系师傅**：声望 100 可拜师，习得独家功法

---

## 8. 派系切换

- **加入**：目标派系声望重置为 25，其他派系 -10
- **离开**：原派系声望 -50，玩家身份变为散修
- **叛逃到对立派系**：原派系 -100，新派系 +25

---

## 9. UI

**派系列表页**：每个派系一行，显示名称、当前等级、称号、声望数值。

**派系详情页**：派系描述、声望进度条、福利列表、任务入口、同门列表（声望 75 后解锁）。

---

## 10. 跟其他系统交互

| 系统 | 交互 |
|---|---|
| 功法 | 派系专属功法需对应声望 |
| 装备 | 派系商店提供高阶法宝 |
| 道侣 | 部分道侣有派系归属，声望影响道侣态度 |
| 弟子 | 派系影响弟子的初始道心 |
| 因果 | 善恶值影响派系声望变化 |
| 战斗 | 派系声望影响同派系 NPC 敌友判定 |
