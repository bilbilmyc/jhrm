# 世界事件 (World Event) 设计

> **Date**: 2026-06-25
> **Status**: DEFERRED to 化神+ (per `docs/decisions.md` 决定 #15：字段保留 forward-compat，6 事件类型/触发/modifier 延后实现)
> **Unlock**: 化神期

---

## 1. 概述

世界事件是影响整个游戏世界的全局事件（虽然单玩家，但世界状态会改变）。化神期解锁（玩家足够强能影响世界走向）。事件持续数天到数月。

---

## 2. 数据模型

```dart
class WorldEvent {
  String id;
  String name;
  EventType type;
  int startYear;                // 修真 ladder 年
  int durationDays;             // 持续天数
  Map<String, double> modifiers;  // 全局 modifier
  List<String> triggerIfIds;    // 触发的 IF 段
  EventStatus status;
}

enum EventType {
  灵潮,      // 灵气爆发，修炼速度 +50%
  魔劫,      // 魔道大兴，战斗奖励 +30%
  仙门,      // 上古仙门开启，可探索
  妖王,      // 妖王出世，Boss 出现
  天灾,      // 天劫降临，修炼受阻
  论道,      // 高人论道，可参与提升道心
}

class WorldState {
  WorldEvent? activeEvent;
  List<WorldEvent> pastEvents;
  Map<Faction, int> factionMoods;  // 派系好感 -100 ~ +100
  int year;                     // 修真 ladder 年
  Season season;
}
```

## 3. 六种事件类型

### 3.1 灵潮

- 触发：每 5-10 年随机发生
- 持续：1-3 月（修真 ladder 时间）
- 效果：所有修炼 +50% 修为速度
- 触发 IF：灵气异象，可选择「采补」或「保守」

### 3.2 魔劫

- 触发：玩家杀魔道 NPC 过多时概率触发
- 持续：6 月
- 效果：魔宗相关掉落 +30%，正道相关掉落 -20%
- 触发 IF：魔道大兴事件，可选「加入」或「抵抗」

### 3.3 仙门大开

- 触发：玩家拥有灵宝以上法宝 + 50 年一次随机
- 持续：1 月
- 效果：可探索仙门获取稀有功法、灵宝
- 触发 IF：仙门入口 IF，可选择「探查」「入内」「不涉」

### 3.4 妖王出世

- 触发：每 20-30 年随机发生
- 持续：长期（妖王被打败或离开）
- 效果：妖族相关 +30% 掉落，野外战斗 +20% 难度
- 触发 IF：妖王现身 IF，可选择「挑战」「结盟」「躲避」

### 3.5 天灾

- 触发：玩家因果大恶 + 随机
- 持续：1-7 天
- 效果：修炼效率 -50%，野外战斗 +30% 难度
- 触发 IF：天劫预警 IF，可选择「祈禳」「硬抗」「迁避」

### 3.6 论道

- 触发：玩家境界达化神 + 随机
- 持续：长期
- 效果：可参与论道提升道心与声望
- 触发 IF：论道邀请 IF，可选择「参与」「旁观」「拒绝」

---

## 4. 触发条件

时间触发：游戏内每 N 年随机检查
状态触发：玩家特定行为累积
剧情触发：完成特定 IF 段后概率触发

---

## 5. UI

**世界状态页**：
- 当前年份与季节
- 当前事件 + 持续 modifier
- 过去事件 log（最近 10 个）

**事件详情页**：
- 事件描述
- 全局 modifier 列表
- 触发 IF 段入口

## 6. 跟其他系统交互

| 系统 | 交互 |

| 功法 | 灵潮期间修炼速度加成 |
| 装备 | 仙门期间可获稀有法宝 |
| 灵兽 | 妖王期间掉落增加 |

| 声望 | 论道事件影响派系声望 |
| 弟子 | 事件期间弟子效率变化 |
| 道侣 | 事件可触发道侣相关 IF 段 |
