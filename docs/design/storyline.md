# 剧情设计 (Storyline Design)

> Slice 21. Authored autonomously — 道心之路：天道将倾，修者何从。

## 中心命题 (Central Premise)

**天道式微 (The Heavenly Dao is decaying).**

玩家是一位误入修真的凡人。游戏开始时，天道已经存在了很久，每一层的灵气都在以不同的方式崩坏：

| 位面 | 灵气之疾 | 隐喻 |
|---|---|---|
| 凡界 | **稀薄** — 灵气淡到几乎不可察 | 修真正在被世人遗忘 |
| 灵界 | **斑驳** — 灵气混有杂质，吸之反伤 | 资源争夺导致互相污染 |
| 仙界 | **凝固** — 灵气凝滞，老仙不愿飞升 | 秩序僵化，新人无路 |
| 神界 | **逆流** — 灵气反向流动，神位失序 | 终极之位的合法性被质疑 |

**核心冲突：天道 vs 道心。**
天道讲秩序与平衡；道心是个人的选择。玩家每升一个位面，就要在「修天道」与「修自己」之间做一次抉择——这条选择链的累积方向，决定飞升后的结局。

## 道心 (Heart Path) 内涵

修真者的内在取向，五维向量（decisions.md #1）。每一条道对应一种面对天道崩坏的态度：

- **剑道**：以个人之力斩断一切阻碍。信奉"我即天道"。
- **王道**：修补秩序，让万物各归其位。信奉"我承天道"。
- **魔道**：天道已坏，不如我取而代之。信奉"我夺天道"。
- **隐道**：天道如何，与我何干。信奉"我避天道"。
- **无道**：天道与道心皆是执，俱当破之。信奉"我非天道"。

## 关键抉择 (Key Decisions)

每个位面都有一个"锚点"抉择，显著推动道心向量：

### 炼气期 (凡界) — 山门老者

老修士在山门等候多时，见到玩家后问："你为何上山？"

| 选项 | 道心变化 | 叙事后果 |
|---|---|---|
| "求长生" | 剑道+1, 王道+1 | 老者授以吐纳基础，提到山门"很久没人来过了" |
| "求力量" | 剑道+2 | 老者神色微变，授以剑入门，暗示"力量是把双刃剑" |
| "求庇护家人" | 王道+2, 魔道+1 | 老者叹气，授以护身咒，提及"凡界最近灵气很淡" |
| "不知道" | 隐道+2, 无道+1 | 老者沉默良久，授以冥想，曰"不知道也是答案" |

### 筑基→金丹 (灵界) — 灵草谷妖精

妖精本是火鬃狐，修炼千年化形。她说灵界的灵气被某种东西污染了，她能感知到源头，但需要玩家帮助。

| 选项 | 道心变化 | 叙事后果 |
|---|---|---|
| "帮她净化源头" | 王道+3, 剑道+1 | 玩家消耗寿元净化，但妖精说"这只是表象" |
| "帮她夺取源头" | 魔道+3 | 玩家获得邪物，妖精说"你我都回不了头了" |
| "让她独自承担" | 隐道+2, 无道+1 | 玩家离开，妖精说"这条路我早该自己走" |
| "问她为何不自己动手" | 剑道+1, 王道+1 | 妖精讲述自己过去的失败，玩家协助但需再抉择 |

### 元婴→化神 (仙界) — 王道遗臣

一位旧朝遗臣在仙界边缘徘徊，他拒绝飞升已经三千年。他说他知道天道崩坏的原因。

| 选项 | 道心变化 | 叙事后果 |
|---|---|---|
| "支持他复位" | 魔道+2, 王道+1 | 玩家获得旧朝信物，遗臣说"你会需要它的" |
| "劝他放下" | 王道+2, 隐道+1 | 遗臣交出残卷，含部分天机，黯然离去 |
| "夺他的位置" | 魔道+3, 剑道+1 | 战斗，遗臣败，玩家获得他的道号与旧怨 |
| "问他为何还不飞升" | 隐道+2, 无道+1 | 遗臣讲述真相：飞升后看到的并非天道本体，而是其镜像 |

### 化神→大乘 (神界) — 隐道仙人

一位早已不存在的隐者，留在神界作为最后的守望者。他不问玩家来意，只问一个问题。

| 选项 | 道心变化 | 叙事后果 |
|---|---|---|
| "天道可修" | 王道+3 | 仙人赠以修补之器，曰"它需要你付出" |
| "天道当立" | 魔道+2, 剑道+1 | 仙人赠以破立之器，曰"它需要你敢" |
| "天道与我何干" | 隐道+3 | 仙人沉默，赠以"归"字牌，曰"它送你回去" |
| "何为天道" | 无道+3 | 仙人化光，遗下"无字碑"，曰"它送你开始" |

### 大乘→飞升 (终局) — 五道抉择

玩家在神界深处找到天道的入口。五个选项对应五条道：

| 选项 | 道心变化 | 结局分支 |
|---|---|---|
| "以剑入道" | 剑道+5 | 剑仙 |
| "以魔代天" | 魔道+5 | 魔尊 |
| "以王承道" | 王道+5 | 圣王 |
| "以隐避世" | 隐道+5 | 散仙 |
| "破道而立" | 无道+5 | 天地客 |

## 主角克制 (Protagonist Restraint)

修真题材最易陷入"金手指爽文"。本作主角的克制：
- **寿元有限** — 1200 月（100 年）封顶，每闭关 -1 月
- **道心冲突** — 五个向量会相互制衡（剑道+魔道不会同时高）
- **不可逆** — 每次抉择都是真选择，存档只能"重置"而不能"回滚"
- **死法隐晦** — 无 Game Over，寿元耗尽时主角自然陨落，但道心向量在身后仍起作用

# v0.2 — 道侣同行 (Dao Companion Arc)

> Slice 39. v0.1 修真者的道是孤独的道；v0.2 让修真者有同行者。

## 中心命题 (Central Premise)

**道侣是"道之镜像"，不是装备。**

玩家在 灵草仙境 IF（slice 37）选择「我欲与君同行」时，玉箫成为玩家的道侣。玉箫有自己的 道心倾向（隐道）、自己的 善恶（略正）、自己的 话语方式（短句、问句、留白）。

修真者与道侣不是「主仆」或「恋人」，是「同行者」——互相影响，互相照见。

## 道侣机制 (Mechanics)

| 行为 | 影响 |
|---|---|
| 共振 (Resonance) | 玩家选的道 与 道侣的 道心倾向 一致时，该道 +1（来自道侣的默许）|
| 逆冲 (Discord) | 玩家选的道 与 道侣倾向 反方向时，该道 -1（来自道侣的不安）|
| 同在 (Presence) | StatusBar 显示道侣名 + 倾向 chip |
| 反馈 (Voice) | 部分 IF 的选项旁出现道侣的"second voice" flavor（不影响选道，只增世界感）|
| 飞升 (Ascension) | 飞升时，道侣有自己的选择：随 / 留 / 道消 |

## 玉箫 — 修真游戏唯一道侣 (v0.2)

| 字段 | 值 |
|---|---|
| 名 | 玉箫 |
| 种 | 玉狐修炼化形 |
| 道心倾向 | 隐道 +3 |
| 善恶 | 略正 (+2) |
| 话语 | 短句、问句、留白。少用动词。|
| 修真 | 不修杀伐，不修权术；唯修心。|
| 一句台词 | "你问何？" |

**道侣的影响 (v0.2 mechanic)**
- 玩家选 隐道 选项：+1 隐道 (共振) — 玉箫的默许
- 玩家选 剑道 选项：-0 (中道，玉箫不置可否)
- 玩家选 王道 选项：-0
- 玩家选 魔道 选项：-1 (逆冲) — 玉箫的不安
- 玩家选 无道 选项：+0 (中道，玉箫不置可否)

这是 v0.2 的"道侣影响" — 不替玩家选，只在最终数值上加 1 / -1。

## 道侣的飞升结局 (v0.2 endings)

修真者飞升时，玉箫有 3 个选择：

1. **随** — 玉箫随玩家飞升，化为新的道侣（道侣与玩家共登）
2. **留** — 玉箫选择留在原界（道侣与玩家离散，飞升画面独行）
3. **道消** — 玉箫在飞升的光中散去（道侣以身殉道，只留一句"去也"）

这 3 个选择不影响玩家的结局（剑仙/魔尊/圣王/散仙/天地客），但影响"飞升画面"的道侣身影。

## 修真小品 (道侣同行 v0.2)

修真小品是道侣的"小声音"，不改变剧情走向，只在选择旁出现道侣的 flavor second voice。

例（凡界·集市）：

> 玩家：「跃入水中，救之」
> 玉箫：（在身后）"......水凉。"
> 玩家：救之

修真小品是 道侣在场的小细节，不影响选道数值。

## 不做 (v0.2 anti-scope)

- **不做** 复杂道侣系统（多个道侣、道侣技能树、道侣装备）
- **不做** 道侣影响选道数值以外的任何东西（道侣不挡刀、不加血、不给钱）
- **不做** 道侣情感系统（"道侣好感度"是庸俗化的同人设定）
- **不做** 道侣专属结局（道侣的飞升是"离/留/殉"，不构成新结局）

- **不做** 多周目差异化（每局都是独立的天道）
