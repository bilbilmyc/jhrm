# 转世/夺舍/轮回 (Reincarnation) 设计

> **Date**: 2026-06-25
> **Status**: DEFERRED to 大乘+ (per `docs/decisions.md` 决定 #15：字段保留 forward-compat，飞升/兵解/夺舍 延后实现)
> **Unlock**: 大乘期

---

## 1. 概述

大乘 9 触发「终局」事件，玩家选择三种路径之一：

- **飞升**：修真成道，飞升离去。飞升 IF 段 → 道心 → 4-6 ending（per ADR-0008）
- **兵解**：肉身死，元神出。转世重生，保留部分修为 + 道心印记
- **夺舍**：占据他人肉身。重修真，保留功法 + 部分修为

支持多周目玩法（修真完成 → 新存档 → 重新开始，继承部分内容）。

## 2. 数据模型

```dart
class ReincarnationState {
  int currentLife;             // 第几世
  List<PastLife> pastLives;    // 过去世记录
  PastLifeCarryover carryover; // 本世继承的内容
  bool isInReincarnation;
}

class PastLife {
  int lifeNumber;
  String daoName;              // 修士名号
  HeartPath finalHeart;        // 最终道心
  Karma finalKarma;            // 最终因果
  Realm maxRealm;              // 最高境界
  List<String> titles;         // 称号
  String causeOfDeath;         // 死因
}

class PastLifeCarryover {
  List<Technique> inheritedTechniques;  // 继承的功法
  Element? rootBonus;                    // 灵根偏移
  int karmaBonus;                        // 因果偏移
  HeartPath? heartImprint;               // 道心印记
  String pastLifeTitle;                  // 过去称号
}
```

## 3. 三种终局

### 3.1 飞升

| 字段 | 内容 |
|---|---|
| 触发 | 大乘 9 突破 |
| 流程 | 飞升 IF 段 → 道心判定 → 4-6 ending |
| 结局 | 游戏结束（保存到 ending 修真）|

修真机制：
- 触发：玩家选择「飞升」
- 流程：飞升 IF 段，4-6 个选择影响最终身份
- 结局：游戏结束，保存到 ending 列表

### 3.2 兵解

| 字段 | 内容 |
|---|---|
| 触发 | 大乘 9 修真失败 |
| 流程 | 兵解 IF 段 → 转世 → 重生 |
| 结局 | 修真完成（保留部分修为）|

修真机制：
- 触发：渡劫失败 + 玩家选择「兵解」
- 流程：兵解 IF 段，元神出窍
- 结局：转世重生，保留部分修为 + 道心印记

### 3.3 夺舍

| 字段 | 内容 |
|---|---|
| 触发 | 飞升失败 + 发现濒死修士 |
| 流程 | 夺舍 IF 段 → 夺舍成功 → 重生 |
| 结局 | 修真完成（保留功法 + 部分修为）|

修真机制：
- 触发：发现濒死修士
- 流程：夺舍 IF 段，道德判定
- 结局：夺舍成功，获得新肉身与身份

## 4. 转世继承机制

继承内容：

| 项目 | 来源 | 数量 |
|---|---|---|
| 功法 | 已学功法 | 1-3 个 |
| 灵根偏移 | 原灵根 | +10% 同元素亲和 |
| 因果偏移 | 原因果 | ±5 |
| 道心印记 | 原道心 | 弱化版 |
| 过去称号 | 称号系统 | 1 个显示 |

不继承内容：修为、装备、灵兽、弟子、道侣、金钱。

---

## 5. 多周目模式

每完成一世的修真解锁多周目：

| 周目 | 触发 | 模式 |
|---|---|---|
| 第 1 世 | 飞升完成 | 普通游戏 |
| 第 2 世 | 第 1 世飞升 | NG+（继承 1 项）|
| 第 N 世 | 第 N-1 世飞升 | NG+N（继承更多）|

每完成一周目，难度提升（敌人 +10% 属性），但继承加成也更多。

## 6. UI

### 6.1 转世 修真面板

```
[转世 第 N 世]

前世：道号
道心：最终方向
因果：最终值
最高境界：最高修为

继承内容：
  · 功法：玄清剑典、灵火诀
  · 灵根偏移：金 +10%
  · 因果偏移：+5
  · 道心印记：剑道（弱）
  · 过去称号：「千年老怪」
```

### 6.2 转世操作

可用操作：
- 开始新一世（保留继承）
- 重新随机（不保留）
- 切换为前世（读取旧存档）
